# AssetScan AI – Arba Minch University Asset Management System

**AssetScan AI** is a modern, intelligent asset tracking and management web application built for **Arba Minch University** (AMU).  
It helps university staff, departments, and administration to efficiently register, track, assign, and monitor university assets — with every asset clearly linked to its **owner** (responsible person, department, or unit).

This project is built with [Next.js 15](https://nextjs.org) (App Router), Tailwind CSS, Chakra UI / Radix UI components, Clerk authentication, MongoDB/Mongoose, and AI-powered features (via Google Generative AI).

## Key Features
- Register and categorize university assets (computers, lab equipment, furniture, vehicles, etc.)
- Assign assets to **owners** (staff members, lecturers, departments, offices)
- Track asset location, status, condition, and maintenance history
- Search, filter, and generate reports (PDF/Excel export possible in future)
- Role-based access (admin, department head, store keeper, viewer)
- Mobile-responsive dashboard with charts and real-time updates
- AI assistance for asset description generation, anomaly detection, etc. (optional)
